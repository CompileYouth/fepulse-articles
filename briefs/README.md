# Briefs

这个目录用于存放“不进入 FEPulse 主文章流程”的内容简报。

使用规则：

- 第一层只放你还没看完的简报 `.md`
- `raw/` 存放这些简报对应的原始字幕
- `archived/` 存放你已经看过的简报

默认流程：

1. 新扫描内容先落到 `ai-interview-archive-data/`
2. 初筛后：
   - 适合 FEPulse 的 raw 字幕移到 `fepulse-reader-picks/`
   - 不适合长文但值得了解的 raw 字幕移到 `briefs/raw/`
3. `briefs/raw/` 中的字幕整理成约 500 字简报，放到 `briefs/` 第一层
4. 你读完之后，可手动把对应简报移到 `briefs/archived/`
