# FEPulse Reader Picks

This directory is not the raw archive.

It is the second-pass selection layer for FEPulse readers.

Goals:

- narrow `ai-interview-archive-data/` down to items that are more likely to interest FEPulse readers
- reduce noise from first-hand AI content that is still not a strong fit for programmer readers
- create a smaller, sharper candidate pool for article writing

Suggested contents:

- `FEPulse-reader-profile.md`
- `<batch-id>.md`: the selection result for a specific scan batch
- cleaned transcript `.txt` files that were selected as strong fits

Prefer items about:

- programmers
- code
- AI coding
- engineering workflow
- developer tools
- builder interviews with practical engineering value
